#Tabuleiro

descrição da atividade:

A entrada para seu programa é composta de duas linhas. Note que o tamanho do tabuleiro pode variar (2, 3, 4, 5 linhas, por exemplo) . A primeira tem a posição inicial do tabuleiro. A segunda, um string com uma sequência de movimentos (d/u/l/r). Por exemplo:

8 6 5 4 0 1 7 3 2

udrrdurl

A saída deve apresentar cada uma das configurações após cada um dos movimentos. E no final deve haver uma verificação se a última configuração representa ou não a solução. Implemente uma classe para representar o tabuleiro e uma classe para o programa principal. A implementação deve ser em Java.
